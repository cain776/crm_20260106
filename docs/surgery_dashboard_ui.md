# 수술실 대시보드 UI 텍스트 와이어프레임

이 문서는 [수술실 운영 효율화 대시보드 기획서](./surgery_dashboard_plan.md)를 바탕으로 한 텍스트 기반 UI 설계도입니다.

## 1. [View A] 실시간 수술실 현황판 (모니터용)

* **목적**: 수술방 상태 직관적 확인 및 신속 대응
* **화면 구성**: 전체 화면을 수술방 개수만큼 분할 (예: 4개 룸) + 하단 통합 정보 바

```text
+-----------------------------------------------------------------------+
|  [2024-05-20 14:30:45]                                   [Dashboard]  |
+-------------------+-------------------+-------------------+-----------+
| ROOM 01           | ROOM 02           | ROOM 03           | ROOM 04   |
| [Status: OP]      | [Status: CLEAN]   | [Status: ANES]    | [EMPTY]   |
| ⚫ 녹색 (정상)     | 🟡 노랑 (대기)    | 🔴 빨강 (지연)     | ⚪ 회색   |
+-------------------+-------------------+-------------------+-----------+
|                   |                   |                   |           |
|  환자: 김철수(M/65)|  (청소 중)        |  환자: 이영희(F/70)| (준비됨)  |
|  수술: 백내장(OD)  |                   |  수술: 유리체절제  |           |
|  집도: 김원장      |                   |  집도: 박원장      |           |
|                   |                   |                   |           |
|  [진행 시간]      |  [남은 시간]      |  [지연 시간]      |           |
|  15분 30초        |  03분 00초        |  +12분 20초 (!!)  |           |
|                   |                   |                   |           |
|  Next: 박지성(대기)|  Next: 준비완료   |  Next: 최동수(검사)|           |
+-------------------+-------------------+-------------------+-----------+
| [오늘의 KPI]                                                          |
| 계획: 40건 / 완료: 22건 (55%)  |  평균 회전율: 08분 12초 (목표 <10분)  |
| 첫수술 정시: 3/4 (75%)         |  특이사항: 2시 마취과 지원 필요       |
+-----------------------------------------------------------------------+
```

### 상태별 색상 코딩 (Color Coding)

* **[OP] 수술 중 (Operation)**: 녹색 배경 / 흰색 글씨
* **[CLEAN] 청소/준비 (Turnover)**: 노랑 배경 / 검은 글씨 (5분 초과 시 붉은 테두리 점멸)
* **[ANES] 마취/대기 (Anesthesia)**: 파랑 배경
* **[DELAY] 지연 (Delay)**: 빨강 배경 / 흰색 글씨 / 경고 아이콘 깜빡임

## 2. [View B] 운영 성과 분석 리포트 (PC/태블릿용)

* **목적**: KPI 분석 및 주간/월간 트렌드 확인
* **레이아웃**: 좌측 네비게이션 + 우측 대시보드 그리드

```text
+----------------+------------------------------------------------------+
| [ MENU ]       |  기간: [ 2024-05-01 ] ~ [ 2024-05-31 ]  [조회]       |
|                +------------------------------------------------------+
| > 대시보드 홈  |                                                      |
|   - 시간 관리  |  [KPI 요약 카드]                                     |
|   - 자원 현황  |  +----------------+  +----------------+  +---------+ |
|   - 재무 분석  |  | 평균 회전율    |  | 가동률 (Block) |  | 정시시작| |
|                |  |    9분 30초    |  |      88%       |  |   92%   | |
| > 수술 스케줄  |  | (▼ 30초 개선)  |  | (▲ 2% 상승)    |  | (목표달성)| |
| > 의료진 관리  |  +----------------+  +----------------+  +---------+ |
| > 설정         |                                                      |
|                +------------------------------------------------------+
|                |                                                      |
|                |  [A. 시간 효율성 차트]   (막대+꺾은선 그래프)        |
|                |  |       |---|           Turnover Time (분)        |
|                |  |   |---|   |---|       ● 목표 (10분)             |
|                |  |---|   |   |   |---|                             |
|                |  1주  2주  3주  4주                                |
|                |                                                      |
+----------------+--------------------------+---------------------------+
|                |                          |                           |
|                |  [B. 지연 사유 분석]     |  [C. 의사별 생산성]       |
|                |  (도넛 차트)             |  (가로 막대 그래프)       |
|                |                          |                           |
|                |      ( 마취 )            |  김원장 20건 ■■■■■     |
|                |   (       ) 30%          |  이원장 18건 ■■■■      |
|                |      ( 준비 )            |  박원장 12건 ■■■       |
|                |                          |                           |
+----------------+--------------------------+---------------------------+
|                |                                                      |
|                |  [D. 알림 및 권장 조치 사항]                         |
|                |  ! [재고] 인공수정체(Toric) 재고가 3개 남았습니다.   |
|                |  ! [일정] 금요일 오후 Room 2 블록이 비어있습니다.    |
|                |                                                      |
+----------------+------------------------------------------------------+
```

### 상세 설명

1. **KPI 요약 카드**: 핵심 지표 3~4개를 최상단에 배치하여 즉시 상태 파악. 전월/전주 대비 증감 표시(▲/▼).
2. **시간 효율성 차트**: 주 단위 또는 일 단위 회전율 변화 추이를 시각화. 목표선(Target Line)을 표시하여 달성 여부 확인.
3. **지연 사유 분석**: 지연의 원인이 무엇인지(마취, 청소, 환자 지각 등) 비율로 표시하여 개선 포인트 도출.
4. **알림/Action Item**: 단순히 현황만 보여주는 것이 아니라, 무엇을 해야 하는지(Call to Action) 텍스트로 제안.
