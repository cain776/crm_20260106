# API 연동 관리 페이지 기능 명세서

## 1. 페이지 개요
- **페이지명**: API 연동 관리
- **파일경로**: `pages/api-integration.html`
- **목적**: 쇼핑몰별 API 연동 설정 및 관리

---

## 2. 기능 버튼 영역 (Action Toolbar)

### 2.1 버튼 구성
| 버튼명 | 아이콘 | 스타일 | 기능 |
|--------|--------|--------|------|
| 등록 | plus | 기본 | 새 API 연동 등록 모달 열기 |
| 수정 | edit | 기본 | 선택된 API 연동 수정 모달 열기 |
| 삭제 | trash-2 | danger (빨간색) | 선택된 API 연동 삭제 |
| 조회 | search | 기본 | 검색 영역 토글 (숨김/출력) |

## 3. 검색 영역 (숨김/출력 토글)

### 3.1 필터 컴포넌트
| 필터명 | 타입 | 옵션 |
|--------|------|------|
| 셀러 | Select (드롭다운) | 전체, 셀러 목록 |
| 쇼핑몰 | Select (드롭다운) | 전체, Qoo10, Rakuten 등 |
| 연동여부 | Select (드롭다운) | 전체, 연동됨, 미연동 |

### 3.2 검색 버튼
| 버튼명 | 기능 |
|--------|------|
| 초기화 | 필터 조건 초기화 |
| 검색 | 필터 조건으로 검색 실행 |

---

## 4. 데이터 테이블 (바로 컬럼명부터 출력)

### 4.1 테이블 컬럼
| 컬럼명 | 너비 | 정렬 | 설명 |
|--------|------|------|------|
| 체크박스 | 40px | 중앙 | 행 선택 |
| 셀러 | auto | 좌측 | 셀러명 |
| 쇼핑몰 | 120px | 좌측 | Qoo10, Rakuten 등 |
| API ID | 150px | 좌측 | API 식별자 |
| 연결상태 | 150px | 좌측 | 연결됨/미연결 뱃지 + Edit 버튼 |
| 생성일시 | 160px | 좌측 | YYYY-MM-DD HH:mm:ss |
| 등록일시 | 160px | 좌측 | YYYY-MM-DD HH:mm:ss |

### 4.2 연결상태 컬럼 상세
- **연결됨**: 녹색 뱃지 (`badge-green`)
- **미연결**: 회색 뱃지 (`badge-gray`)
- **Edit 버튼**: 노란색 버튼, 클릭 시 수정 모달 열기

### 4.3 빈 데이터 상태
- 텍스트: "등록된 API 연동이 없습니다."
- 아이콘: link-2 (연결 아이콘)

## 6. 모달 (팝업)

### 6.1 API 연동 등록 모달 (WMS 선택 모달과 동일한 형태)

**모달 구조:**
```
┌─────────────────────────────────────────────────┐
│  쇼핑몰 선택                                 ✕  │  ← 파란색 헤더
├─────────────────────────────────────────────────┤
│                                                 │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐     │
│   │  Qoo10   │  │ Rakuten  │  │  Shopee  │     │
│   └──────────┘  └──────────┘  └──────────┘     │
│                                                 │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐     │
│   │  Amazon  │  │  eBay    │  │ Shopify  │     │
│   └──────────┘  └──────────┘  └──────────┘     │
│                                                 │
└─────────────────────────────────────────────────┘
```

**쇼핑몰 옵션 버튼:**
| 쇼핑몰명 | 색상 |
|----------|------|
| Qoo10 | 빨간색 (text-red-500) |
| Rakuten | 빨간색 (text-red-500) |
| Shopee | 주황색 (text-orange-500) |
| Amazon | 주황색 (text-orange-500) |
| eBay | 파란색 (text-blue-500) |
| Shopify | 녹색 (text-green-500) |

→ 쇼핑몰 선택 시 API 정보 입력 모달로 이동

### 6.2 API 정보 입력 모달 (쇼핑몰 선택 후)

**모달 구조:**
```
┌─────────────────────────────────────────────────┐
│  API 연동 등록 - Qoo10                       ✕  │  ← 파란색 헤더
├─────────────────────────────────────────────────┤
│                                                 │
│  셀러 *        [셀러 선택 ▼                   ] │
│                                                 │
│  API ID *      [API ID 입력                   ] │
│                                                 │
│  API Key *     [API Key 입력                  ] │
│                                                 │
│  API Secret    [API Secret 입력               ] │
│                                                 │
├─────────────────────────────────────────────────┤
│                        [연결 테스트]  [저장] [취소] │
└─────────────────────────────────────────────────┘
```

| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| 셀러 | Select | ✓ | 셀러 선택 |
| API ID | Input | ✓ | API 식별자 입력 |
| API Key | Input | ✓ | API 인증키 입력 |
| API Secret | Input | - | API 시크릿키 입력 (선택) |

### 6.3 API 연동 수정 모달 (Edit 버튼 클릭 시)

**모달 구조:**
```
┌─────────────────────────────────────────────────┐
│  API 설정                                    ✕  │  ← 흰색 헤더
├─────────────────────────────────────────────────┤
│                                                 │
│                 ID        [gyeolhaus          ] │
│   ┌─────────┐                                   │
│   │  Qoo10  │  Password   [                   ] │
│   │  (로고) │                                   │
│   └─────────┘  API Key    [BhiQeRsAhG2v2Sk... ] │
│                                                 │
│   *패스워드는 저장되지 않습니다.                │
│   *API Key는 B*** 형태의 Key 입니다..           │
│                                                 │
├─────────────────────────────────────────────────┤
│                                        [인증]   │  ← 파란색 버튼
└─────────────────────────────────────────────────┘
```

**모달 특징:**
- 왼쪽에 해당 쇼핑몰 로고 표시 (Qoo10, Rakuten 등)
- 기존 데이터 자동 로드 (ID, API Key)
- Password 필드는 빈 상태 (저장되지 않음)
- 하단 안내 문구 표시
- "인증" 버튼으로 API 연결 테스트 및 저장

| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| ID | Input | ✓ | API 식별자 (기존 값 표시) |
| Password | Input | - | 비밀번호 입력 (저장 안됨) |
| API Key | Input | ✓ | API 키 (기존 값 표시) |

**안내 문구:**
- *패스워드는 저장되지 않습니다.
- *API Key는 B*** 형태의 Key 입니다..

### 6.4 삭제 확인 모달
- 제목: "API 연동 삭제"
- 메시지: "선택한 {N}개의 API 연동을 삭제하시겠습니까?"
- 버튼: 취소, 삭제
