## 🎯 경쟁사 대비 차별화 포인트

### 안과 특화 메뉴

| 메뉴 | 차별화 내용 |
|------|-----------|
| **수술 관리** | 굴절교정/백내장/녹내장/망막 수술 전용 모듈 |
| **검사 관리** | 안과 검사 유형별 분류 및 장비 연동 |
| **소개환자 관리** | VIP 왕관 랭킹 + 소개 보상 정책 |
| **의사별 분석** | 수술 건수/성공률/매출 분석 |

### 경쟁사에 없는 기능

1. ✅ **수술 전용 모듈** - 수술 유형별 워크플로우
2. ✅ **검사 장비 연동** - OCT, 안압계 등 자동 연동
3. ✅ **VIP 왕관 랭킹** - 고객 등급 시각화
4. ✅ **소개환자 보상 관리** - 인센티브 자동 계산
5. ✅ **분원 통합 관리** - 본원/분원 데이터 통합
