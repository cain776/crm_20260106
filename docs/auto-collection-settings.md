# 자동 등록 설정 메뉴 구현 계획

## 1. 개요
자동 수집 스케줄을 설정하여 주문을 자동으로 수집하고 처리하는 설정 페이지

---

## 2. UI 구성요소

### 2.1 자동 수집 활성화
| 요소 | 설명 |
|------|------|
| 제목 | "자동 수집 활성화" |
| 설명 | "이 기능을 켜면 설정된 스케줄에 따라 자동으로 주문을 수집합니다." |
| 컨트롤 | 토글 스위치 (ON/OFF) |

---

### 2.2 수집 스케줄 설정
| 요소 | 타입 | 옵션 |
|------|------|------|
| 아이콘 | 📅 캘린더 | - |
| 수집 기간 | Date Range Picker | 시작일 ~ 종료일 |
| 수집 주기 | Select 드롭다운 | 1분마다, 5분마다, 10분마다, 30분마다, 1시간마다 |
| 요일 선택 | 버튼 그룹 | 월, 화, 수, 목, 금, 토, 일 (다중 선택) |

#### 요일 버튼 상태
- **선택됨**: 파란색 배경 (#2563eb) + 흰색 텍스트
- **미선택**: 흰색 배경 + 회색 테두리

---

### 2.3 수집 대상 쇼핑몰
| 요소 | 설명 |
|------|------|
| 아이콘 | 🛒 쇼핑카트 |
| 제목 | "수집 대상 쇼핑몰" |
| 컨트롤 | 토글 버튼 그룹 (다중 선택) |

#### 쇼핑몰 목록
- Qoo10
- Rakuten
- PlayAuto
- NHN
- (추가 가능)

#### 버튼 상태
- **선택됨**: 파란색 배경 + 흰색 텍스트
- **미선택**: 흰색 배경 + 회색 테두리

---

### 2.4 수집 후 자동 전송
| 요소 | 설명 |
|------|------|
| 아이콘 | 👤 사용자 |
| 제목 | "수집 후 자동 전송" |
| 컨트롤 | 라디오 버튼 카드 |

#### 옵션
1. **자동 전송** (기본 선택)
   - "수집 완료 후 '배송준비' 상태로 자동 전송합니다."
   - 선택 시: 보라색/파란색 테두리 강조

2. **수동 전송**
   - "수집만 수행합니다. (전송은 수동으로 진행)"
   - 선택 시: 보라색/파란색 테두리 강조

---

### 2.5 하단 액션 버튼
| 버튼 | 스타일 | 동작 |
|------|--------|------|
| 취소 | Secondary (회색 테두리) | 설정 취소, 이전 페이지로 |
| 테스트 | Secondary (회색 테두리) | 현재 설정으로 테스트 수집 실행 |
| 저장 | Primary (파란색 배경) | 설정 저장 |

## 4. 스타일 가이드

|------|-----|

| 카드 패딩 | 24px |

| 라벨-컨트롤 간격 | 12px |
